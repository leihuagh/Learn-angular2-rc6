# Learning source from 
angular2-rc6 https://github.com/AcademiaBinaria/angular2-RC6

## start package includes:
-Angular 2 v2.0.0
-Webpack
-Webpack-dev-server
-SASS
-TypeScript
-Bootstrap 4.0.0-alpha.4
-JQuery v3.1.1
-Font Awesome

### Installation

```
$ npm install
$ npm start
```

#### When installation gets failure, npm install again.
